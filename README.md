# Crypto Trading Scripts

This repository contains simple Python scripts I built to develop my understanding of systematic trading, risk management, and strategy backtesting in the digital assets market.

## Scripts

### 1. Moving Average Crossover Backtest
- Simple SMA(50/200) crossover strategy
- Uses historical price data (CSV)
- Tracks basic performance (PnL)

### 2. Position Sizing Calculator
- Calculates position size based on:
  - Account balance
  - Risk per trade (%)
  - Stop loss distance (pips / price difference)

## Purpose
The goal of these scripts is to:
- Practice rule-based trading logic
- Improve risk management discipline
- Transition towards more systematic and data-driven trading approaches
